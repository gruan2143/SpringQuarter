Website Style Guide – Gang Ruan

 

Overall Formatting

 

The overall site colors are greenish gray. 

 

Font colors are black, blue, brown, red.

 

The general font-family for the site is font-family: Georgia, Times, serif; font-size: 12pt

 

Site Header / Branding 

 

The header area font is:

 

font-family: Arial, Helvetica, sans-serif; font-size: 1.8em; font-weight: bold; color: black

Navigation Links 

 

Site navigation uses the font-family: Georgia, Times, serif; font-size: 12pt.

                                         font color: blue with underline.

 

Content Section 

 

The webpage content section uses padding and margin of 10px. 

 

Site Footer 

 

The footer has a background color of greenish gray.

Font-size: small

Font color: pink
